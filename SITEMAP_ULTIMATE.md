## 🌳 CHI TIẾT CẤU TRÚC PHÂN NHÁNH CẤP 3 (SUB-BRANCHES)

### 1. 📦 QUẢN LÝ SẢN PHẨM (Trong Admin CMS)
*   **Thêm sản phẩm mới:** Form thiết lập thông số sản phẩm, tải ảnh và kích hoạt thuộc tính cá nhân hóa.
*   **Chỉnh sửa sản phẩm:** Chỉnh sửa thông tin, giá, kho hàng của sản phẩm hiện tại.

### 🚚 2. QUẢN LÝ ĐƠN HÀNG (Trong Vận hành Admin)
*   **Duyệt đơn hàng:** Cập nhật trạng thái đơn (Chờ duyệt, Đang giao, Đã hoàn thành).
*   **Xử lý thêu khắc:** Tiếp nhận chi tiết yêu cầu cá nhân hóa (thêu chữ lên gấu, khắc laser).

### ⚙️ 3. CÀI ĐẶT HỆ THỐNG (Trong Cấu hình Admin)
*   **Tích hợp API Lazada:** Đồng bộ sản phẩm, tồn kho và đơn hàng tự động từ sàn Lazada.
*   **Tích hợp API Tiki:** Kết nối và đồng bộ dữ liệu với sàn Tiki.
*   **Tích hợp API TikTok Shop:** Tích hợp quản lý đơn hàng thời gian thực từ TikTok Shop.
